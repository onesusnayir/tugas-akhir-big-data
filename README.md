# Dataset Overview

Dataset ini berisi informasi pembelian minuman beralkohol jenis spirits oleh pemegang lisensi minuman keras Kelas “E” di Iowa, mulai dari 1 Januari 2012 hingga Agustus 2023. Data yang tersedia mencakup jenis produk, tanggal pembelian, serta informasi penjualan pada tingkat toko, sehingga dapat digunakan untuk menganalisis total penjualan minuman beralkohol tertentu di setiap toko. Lisensi minuman keras Kelas “E” sendiri adalah izin khusus yang diberikan kepada toko atau bisnis untuk menjual minuman beralkohol dalam kemasan tertutup, seperti botol minuman keras yang dijual di supermarket, minimarket, atau toko minuman, bukan untuk langsung dikonsumsi di tempat seperti di bar atau restoran.

Berikut adalah dataset yang kami gunakan: https://www.kaggle.com/datasets/csafrit2/iowa-liquor-sales/data
